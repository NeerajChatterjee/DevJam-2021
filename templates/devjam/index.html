Features We Offer

We built a college website where student can access multiple features in a single place. This website supports speech recognition, which saves time and helps every student.

Search Engine

It takes input as text or voice. It uses Wikipedia and WolframAlpha to get the correct result.

Note Maker

It converts the voice input to text and then stores the content into a file so that the student in a doc format can access it.

File Extension Converter

It converts the given image file(jpg/jpeg/png) and doc/docx file to pdf format.

MNNIT Connect Extension

It has a timer that shut down the tabs automatically. Students can enter the time and the tabs that they want to close after a certain period—supported on Google chrome. Download it and read the guide.

MNNIT MeetCalm Extension

It hides the sensitive data by making a specific part blur. Students/Faculties can select the areas to blur and use them while doing screen sharing. Download it and read the guide.
